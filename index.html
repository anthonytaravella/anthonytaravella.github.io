My Design Process

  • 1 · Research / Game IdeationStudy the genre. Every game needs a core gameplay loop that will keep players playing.
  • 2 · PrototypeBuild the core loop fast and ugly. If it isn't fun in a graybox, it won't be fun looking cool.
  • 3 · BuildSimple (not really), build the game.
  • 4 · PlaytestPlaytest Playtest Playtest Playtest Playtest
  • 5 · Ship / MarketRelease planning, launch.
  • 6 · StudyRead the analytics, find the biggest problem, fix it, repeat.
